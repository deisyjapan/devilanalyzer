１．このソフトは？
　　D2真・女神転生リベレーションの悪魔情報を検索するフリーソフトです。
　　あるスキルを所持している悪魔を検索したり、ある耐性を持っている悪魔を検索したりできます。
    検索結果のスキル名をダブルクリックすることでスキルの詳細を、悪魔名をダブルクリックすることで
　　その悪魔を合体で作成するための悪魔の組み合わせを表示します。
    OSがWindows10 64bit/Windows7/32,64bitのいずれかで、かつ.NetFramework4.6.1で動くように
　　作られています。
